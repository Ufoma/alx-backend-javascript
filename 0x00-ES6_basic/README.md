# ES6 Project
==============

## Description 
-----------

A basic template for ES6 projects, configured with Jest, Babel, and ESLint.


### Features 

#### Key Features

*   ES6+ syntax support 
*   Jest testing framework 
*   Babel transpilation 
*   ESLint code linting 


## Installation 
---------------

### Dependencies 

#### Required Versions

*   Node.js (≥14.17.0)
*   npm (≥6.14.13)


### Setup 

```bash
npm install
